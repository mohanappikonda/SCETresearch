# SCET Research Portal - Frontend

This is the frontend of the SCET Research Profile Management Portal, built with **React (Vite)** and styled with **Material UI** and **Bootstrap**.

## 🚀 Features

- **Dashboard**: Institutional analytics and research metrics.
- **Faculty Profiles**: Detailed view of faculty research output.
- **Management**: Easy-to-use forms for updating publications, patents, and projects.
- **Branding**: Institutional branding for Swarnandhra College of Engineering & Technology.

## 🛠️ Setup Instructions

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Create a `.env` file (or set environment variables) with:
    ```env
    VITE_API_BASE_URL=https://your-backend-url.com/api
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```

## 🌐 Deployment

Designed for seamless deployment on **Vercel** or **Netlify**.
